# 个人博客管理系统

### 本地环境 导入数据库文件 blog.sql 

* 数据库账号密码修改 在 config目录下面的 db.js,自行修改即可

* 安装nodejs环境 http://nodejs.cn/ 自行下载

* 运行项目 进入blog目录下面 node ./bin/server 即可运行项目

* 导入blog.sql文件，获取已有的数据库信息
